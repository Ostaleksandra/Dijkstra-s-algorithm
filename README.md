# Dijkstra Algorithm (Алгоритм Дейкстры)

В данном проекте реализован **алгоритм Дейкстры** на языке **Python** для поиска кратчайшего пути в графе.

Алгоритм применяется для вычисления минимального расстояния от одной вершины графа до всех остальных.
В качестве примера используется модель **сети станций метро**, где:

* станции — вершины графа
* переходы между станциями — рёбра
* числа — расстояния между станциями

---

# Используемый граф

Граф представлен в виде словаря Python:

G = {
    "Адмиралтейская": {"Садовая": 4},
    "Садовая": {"Сенная площадь": 3, "Спасская": 3, "Адмиралтейская": 4, "Звенигородская": 5},
    "Сенная площадь": {"Садовая": 3, "Спасская": 3},
    "Спасская": {"Садовая": 3, "Сенная площадь": 3, "Достоевская": 4},
    "Звенигородская": {"Пушкинская": 3, "Садовая": 5},
    "Пушкинская": {"Звенигородская": 3, "Владимирская": 4},
    "Владимирская": {"Достоевская": 3, "Пушкинская": 4},
    "Достоевская": {"Владимирская": 3, "Спасская": 4}
}

---

# Как работает алгоритм

Алгоритм Дейкстры выполняет следующие шаги:

1. Выбирается начальная вершина (стартовая станция).
2. Для всех вершин устанавливается расстояние **∞ (бесконечность)**.
3. Для стартовой вершины расстояние устанавливается **0**.
4. Выбирается вершина с **минимальным расстоянием** среди непосещённых.
5. Проверяются соседние вершины и обновляются расстояния.
6. Вершина помечается как посещённая.
7. Процесс повторяется, пока не будут обработаны все вершины.

---

# Пример задачи

Начальная станция:

Адмиралтейская

Конечная станция:

Владимирская


Программа вычисляет:

* минимальные расстояния до всех станций
* кратчайший путь между выбранными станциями
